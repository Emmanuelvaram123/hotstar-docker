# 🎬 Hotstar Clone - Full Stack Prototype

A high-fidelity streaming platform clone featuring a modern UI, Tollywood (Telugu) content focus, and a simulated microservices architecture using Java and Python logic.

## 🚀 Features
- **Modern UI/UX**: Dark-themed interface with horizontal scrolling carousels and smooth hover effects.
- **Tollywood Edition**: Curated sections for latest Telugu movies (e.g., *Game Changer*, *Thandel*) and popular web series.
- **Responsive Hero Section**: Dynamic featured banner with a clean glassmorphism navigation bar.
- **Cross-Platform Logic**: 
  - **HTML5/CSS3**: Frontend structure and styling.
  - **JavaScript**: Acts as the "Glue Code" to simulate backend data delivery.
  - **Java (Architecture)**: Designed for User Management and API Gateway.
  - **Python (Logic)**: Designed for Recommendation Engines and Data Analytics.

## 🛠️ Technology Stack
| Layer | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | HTML5, CSS3 | Layout, Carousels, & Branding |
| **Logic/Glue** | JavaScript (ES6) | Data rendering & UI Interactivity |
| **Backend** | Java (Spring Boot) | *Conceptual:* Secure API & User Auth |
| **Intelligence** | Python (Pandas/Flask) | *Conceptual:* Movie Recommendation Algorithm |
